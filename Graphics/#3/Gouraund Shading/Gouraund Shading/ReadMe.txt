OBJ 파일은 버전에 따라 여러 사항들을 제공합니다.
모든 사항들을 다 처리하는 프로그램을 만들 순 없기 때문에,
이 프로그램에서는 V, Vn, F 정보가 들어있는 OBJ 파일을 처리하겠습니다.

(또한, 오직 Face는 삼각형으로만 이루어져 있다고 가정합니다.)

	1. 'q' 를 통해 드로우 방식을 바꿀 수 있습니다.
		a. 와이어프레임
		b. 고로쉐이딩
		c. 와이어프레임 + 노멀벡터
	2. w, s, a, d, z, x 키를 이용하여 광원의 위치를 바꿀 수 있습니다.
		w : X + 0.1
		s : X - 0.1
		a : Y + 0.1
		d : Y - 0.1
		z : Z + 0.1
		x : Z - 0.1
	3. r 키를 이용하여 모델을 회전시키거나 멈출 수 있습니다.
		* 회전 각도는 시간에 비례하지 않고 프레임 단위로 움직이므로,
		고로 쉐이딩 드로우때는 느려질 수 있습니다. 정상입니다.
